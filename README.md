## 기술 스텍 및 라이브러리 
- 주요언어: Typescript
- 주요 라이브러리: ReactJS
- 상태관리 : react-query , zustand
- Fake JSON data API: DummyJSON (https://dummyjson.com/)
- 배포 : firebase
- 스타일 : tailwind css
- 그외 : react-router-dom
